
## wordbot API


### Get a single word

`GET https://api.noopschallenge.com/wordbot`

`HTTP 200`

```
{
  "words": [ "quadratically" ]
}
```


### Request Verbs

`GET https://api.noopschallenge.com/wordbot?set=verbs&count=10`

`HTTP 200`

  "words": [
